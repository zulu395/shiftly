## ðŸ“– About Shiftly

Shiftly is a comprehensive staff scheduling and management platform built with Next.js, React, and MongoDB. It provides companies with tools to:

- Schedule and manage employee shifts
- Track employee availability
- Organize company events
- Facilitate team communication
- Monitor analytics and performance

The platform follows a modern 3-layer architecture with server actions, services, and models, ensuring clean separation of concerns and maintainability.
